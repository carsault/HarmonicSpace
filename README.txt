Install python3 :
https://www.python.org/downloads/

Install tkinter :
$ sudo apt-get install python3.6-tk

Install dependencies :
$ pip3 install keras
$ pip3 install tensorflow
$ pip3 install joblib
$ pip3 install tqdm
$ pip3 install pandas
$ pip3 install numpy
$ pip3 install muda
$ pip3 install jams
$ pip3 install pumpp
$ pip3 install sklearn
$ pip3 install matplotlib
$ pip3 install h5py


Put the audio files in the folder « audio »
Warning : If new audio files in « audio » delete pump/ folder and tsne*.txt

Go to the folder DuSapinTsne and run the python script 
$ python3 TsneClassif.py
Note: you can change the dimension (2 or 3) of the Tsne into TsneClassif.py by changing the variable tsnedim = 2 or 3.
